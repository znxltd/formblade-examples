# File Upload Form

A contact form with file attachment support.

## Usage

Replace `contact` in the form action with your FormBlade endpoint ID.

## Important

- The form must have `enctype="multipart/form-data"` for file uploads to work
- Personal plan: up to 1 MB per file
- Pro plan: up to 25 MB per file
- Allowed types: PDF, DOC, DOCX, XLS, XLSX, CSV, TXT, JPG, PNG, GIF, WebP, SVG
- Files are stored securely and downloadable from the FormBlade dashboard
