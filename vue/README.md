# Vue 3 Contact Form

A Vue 3 component using the Composition API that submits to FormBlade.

## Usage

```vue
<script setup>
import ContactForm from './ContactForm.vue';
</script>

<template>
  <ContactForm />
</template>
```

## Features

- Vue 3 Composition API with `ref`
- Reactive form state and loading indicator
- Success message replaces form
- Error handling
